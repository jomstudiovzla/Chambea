# Chambea

App iOS para empleo remoto internacional en español e inglés. Busca vacantes multi-fuente, gestiona tu perfil multimedia y usa IA para mejorar tu empleabilidad.

## Instalar en iPhone

Escanea el código QR:

![Instalar Chambea en iPhone](docs/Chambea-QR-Instalar-iPhone.png)

O abre la guía directamente:

- [Guía móvil de instalación](docs/install.html)
- [Instrucciones completas](INSTALAR-IPHONE.md)

> iOS requiere Xcode en Mac para instalar en desarrollo. TestFlight llegará en una versión posterior.

## Repositorio

![QR Repositorio GitHub](docs/Chambea-QR-GitHub.png)

```bash
git clone https://github.com/jomstudiovzla/Chambea.git
cd Chambea
open Chambea.xcodeproj
```

## Requisitos

- macOS con Xcode 15+
- iPhone con iOS 17+
- Apple ID (gratuito o cuenta Developer)

## Estructura

- `Chambea/` — código fuente Swift (MVVM-C)
- `ChambeaTests/` — tests unitarios
- `docs/` — QR y guía de instalación
- `scripts/` — utilidades (QR, XcodeGen, verificación)
