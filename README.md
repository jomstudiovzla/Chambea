# Chambea

App iOS para empleo remoto internacional en español e inglés. Busca vacantes multi-fuente, gestiona tu perfil multimedia y usa IA para mejorar tu empleabilidad.

## Instalar en iPhone

### 1. Descargar ZIP (Mac)

**[⬇️ Descargar Chambea.zip](https://github.com/jomstudiovzla/Chambea/archive/refs/heads/main.zip)**

Descomprime → lee `LEEME-PRIMERO.txt` → abre `Chambea.xcodeproj` → conecta iPhone → Run ▶

### 2. Guías

- [Guía completa de instalación](docs/GUIA-INSTALACION-IPHONE-COMPLETA.md)
- [Guía móvil (web)](https://jomstudiovzla.github.io/Chambea/install.html)
- [Instrucciones resumidas](INSTALAR-IPHONE.md)

### 3. Código QR

![Instalar Chambea en iPhone](docs/Chambea-QR-Instalar-iPhone.png)

> iOS requiere Xcode en Mac para instalar. TestFlight llegará en una versión posterior.

## Repositorio

![QR Repositorio GitHub](docs/Chambea-QR-GitHub.png)

```bash
git clone https://github.com/jomstudiovzla/Chambea.git
cd Chambea
open Chambea.xcodeproj
```

## Requisitos

- macOS con Xcode 15+
- iPhone con iOS 17+
- Apple ID (gratuito o cuenta Developer)

## Estructura

- `Chambea/` — código fuente Swift (MVVM-C)
- `ChambeaTests/` — tests unitarios
- `docs/` — QR y guía de instalación
- `scripts/` — utilidades (QR, XcodeGen, verificación)
