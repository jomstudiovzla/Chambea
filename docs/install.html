Chambea

Empleo remoto para hispanohablantes

Instalar en tu iPhone (Telefono)

  1. En tu Mac, descarga el proyecto:
    Descargar ZIP
  2. Abre Chambea.xcodeproj en Xcode.
  3. En Signing & Capabilities, elige tu Apple ID como Team.
  4. Conecta tu iPhone Telefono por USB y toca Confiar.
  5. Selecciona tu iPhone arriba y pulsa Run ▶.
